## 연산자 (Operator)

### 대입 연산자

오른쪽 피연산자는 주로 산술식 또는 논리식을 수행하는 표현식, 그 결과는 왼쪽 피연산자인 변수나 상수에 할당

```swift
var x: Int ? // 옵셔널 Int 변수 선언
var y = 10 // 일반 Int 변수 선언, 초기화
x = 10 // 값을 x에 할당, Optional(10)
x = x! + y // x + y의 결과를 x에 할당, Optional(20)
x = y // y의 값을 x에 할당, Optional(10)
```

### 증가 연산자와 감소 연산자

* x의 값을 1 증감

```swift
// 다른 언어와 차이점 없음
x = x + 1 // x변수의 값을 1 증가
x = x - 1 // x변수의 값을 1 감소
```

* 다른 언어와 다른 점

```swift
x++ // x를 1 증가, Swift 3에서 없어짐
x-- // x를 1 감소, Swift 3에서 없어짐
```

### 비교 연산자

```swift
var result: Bool?
var x = 10
var y = 20
result = x < y // True
```

* ==는 값이 같은지를 비교
* ===는 동일한 메모리 주소인지 비교

## 제어문

### For 문

* 조건-증가 for문은 Swift 3에서 없어진 문법임.

```c
for int i = 0; i < 10; i++ {
    printf(i)
}
```

```swift
for i in 0..<10 {
    print(i)
}
```

* Swift에서는 _로 참조체 생략 가능

### 제어문 주의사항

```swift
for i in 1..<10 {
    if i > 5 break
    print(i)
}
// 에러 발생

// if문 다음의 실행 코드가 한줄이라도 괄호를 필수적으로 사용해야함.

for i in 1..<10 {
    if i > 5 {
        break
    }

}
```

#### Where 절

* Switch-Case 문에 부가적인 조건을 추가하기 위해 사용
* 값이 속하는 범위뿐만 아니라 그 숫자가 홀수인지 짝수인지도 검사

```swift
var temperature = 60
switch (temperature) {
    case 0...49 where temperature % 2 == 0:
        print("Cold and even")
    case 50...79 where temperature % 2 == 0:
        print("Warm and even")
    case 80...110 where temperature % 2 == 0:
        print("Hot and even")
    default:
        print("Temperature out of range or odd")
}
```

* 특정 패턴과 결합하여 조건을 추가할 수 있음

```swift
var numbers: [Int] = [1, 2, 3, 4, 5]
for num in numbers where num > 3 {
    print(num)
}
```

#### Fallthrough

* C / C++, Objective-C와 달리 case 문 다음에 break 문을 포함할 필요가 없음
* Swift에서는 일치하는 case 문을 만나면 자동으로 빠져나옴
* case 별로 빠져나가지 않고 아래로 계속 내려가게 하려면 fallthrough 문 사용

```swift
var value = 4
switch (value) {
    case 4:
        print("4")
        fallthrough
    case 3:
        print("3")
        fallthrough
    case 2:
        print("2")
        fallthrough
    default:
        print("1")
}
```

### Swift에서 많이 사용하는 제어문

* 조건문 (if-else)

```swift
let score = 85

if score >= 90 {
    print("A")
} else if score >= 80 {
    print("B")
} else if score >= 70 {
    print("C")
} else {
    print("D")
}
```

* switch 문

```swift
let day = 3

switch day {
case 1:
    print("월요일")
case 2:
    print("화요일")
case 3:
    print("수요일")
case 4:
    print("목요일")
case 5:
    print("금요일")
default:
    print("주말")
}
```

* 반복문 (for-in)

```swift
let numbers = [1, 2, 3, 4, 5]

for number in numbers {
    print(number)
}
```

* while 문

```swift
var count = 0

while count < 5 {
    print("Count is \(count)")
    count += 1
}
```

* repeat-while 문

```swift
var count = 0

repeat {
    print("Count is \(count)")
    count += 1
} while count < 5
```

## 옵셔널 (Optional)

### Int형 vs Optional Int형

* 10
    - Int
        + Int형 값을 저장
        + var x : Int = 10 // x?

* Optional(10)
    - Int?
    - Int!
        + Int형 값을 저장 or 값이 없음 (nil)
        + var y : Int? = 10 // y?
        + var z : Int! // z?

### 강제 언래핑 (Forced Unwrapping)

* 옵셔널 변수에 값이 있으면 옵셔널로 "래핑되었다(Wrapped)"고 함.
* 옵셔널에 래핑된 값은 강제 언래핑 (Forced Unwrapping)으로 풀어줌.

* 옵셔널형 선언: 자료형 뒤 ?
* 옵셔널 언래핑 : 변수명 뒤 !

```swift
var x : Int? // 옵셔널 정수형 변수 x 선언
x = 10

print(x) // Optional(10)
print(x!) // Forced Unwrapping해서 10이 나옴
```
